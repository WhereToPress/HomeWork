Зайти в акаунт GitHub.
Создать внешний репозиторий с назанием group.
- клик "+", New Repository, name - "group" scroll down, click button "Create repository".
Клонировать репозиторий на локальный компьютер.
- git clone https://github.com/WhereToPress/group.git. cd group.
оставит коммит на пустом репозитории
- git commit --allow-empty -m "first commit"
1. На локальном репозитории сделать ветки для: Postman, Jmeter, CheckLists, Bag_Reports, SQL, Charles, Mobile_testing.
- git branch Postman 
- git branch Jmeter 
- git branch Check_lists 
- git branch Bug_Reports 
- git branch SQL 
- git branch Charles 
- git branch Mobile_testing.
2. Запушить все ветки на внешний репозиторий.
- git push origin Postman
- git push origin Jmeter 
- git push origin Check_lists 
- git push origin Bug_Reports 
- git push origin SQL 
- git push origin Charles 
- git push origin Mobile_testing.
or
- git push --all -u origin
3. В ветке Bag_Reports сделать текстовый документ со структурой баг репорта.
- git checkout Bug_reports. touch bug_report.txt. nano bug_report.txt.
4. Запушить структуру багрепорта на внешний репозиторий.
- git add ., git commit -m "Add bug_report.txt", git push -u origin Bug_reports.
5. Вмержить ветку Bag Reports в Main.
- git checkout main. git merge Bug_Reports. :wq
6. Запушить main на внешний репозиторий.
- git push -u origin main.
7. В ветке CheckLists набросать структуру чек листа.
-  git checkout Check_lists. touch check_list.txt. nano check_list.txt.
8. Запушить структуру на внешний репозиторий.
- git add Check_list.txt. git commit -m "Creat check_list.txt in branch Check_lists".  git push -u origin Check_lists.
9. На внешнем репозитории сделать Pull Request ветки CheckLists в main.
- на веб интерфейсе github.com в репозитории group перейти на закладку Pull requests. Creat new pull request. Выбрать ветки слияния (base: main / compare: Check_lists). Creat pull request. Creat pull request. Merge pull request. Confirm merge.
10. Синхронизировать Внешнюю и Локальную ветки Main.
- git checkout main. git fetch. git pull origin main.
